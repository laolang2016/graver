## 关于本部分教程的一些说明

1. 由于 `cmake` 是我写的第一个教程, 所以可能有很多缺点
2. 由于 `windows` 有各种奇怪的问题, 所以我在 `cmake` 教程部分只考虑 `Linux` 系统, 建议 `windows` 用户安装个虚拟机
3. 编辑器就用 `vscode` 即可, 具体的 `vscode` 环境配置参考: [vscode C/C++ 配置](/setup/vscode/a02_base_setting.html#c-c)
4. 由于 `cmake` 入门非常简单, 所以没有把代码放在 `github` 上
5. 暂时不涉及 `vcpkg` 等包管理工具
6. 会对 `cmake` 的命令做出简要解释, 但是详细的解释还请参考官方文档
